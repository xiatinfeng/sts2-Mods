### 关键架构决策

**2026-06-19 Control 根节点 + 手动定位**
- 根节点用 `Control` 而非 `Panel`/`Container`，避免 Container 强制覆盖子节点尺寸
- 卡牌网格弃用 `GridContainer`，改用 `SetPosition` + `SetSize` 手动定位
- 固定卡片尺寸 140×48px，列数动态计算
- 用 `GetTree().CreateTimer(0.05f)` 延时一帧等布局就绪

**2026-06-19 Container 陷阱修复**
- `SizeFlagsHorizontal = ExpandFill` 在 `Control` 父节点下被忽略
- 改用 `SetAnchorsAndOffsetsPreset(FullRect)` 锚定
- 详见 `~/.agents/debug/2026/06-June/2026-06-19-pi-container-trap.md`

**2026-06-19 底部 Dock 修复**
- `bottomDock` 原为 `Panel`（非 Container），内部 `dockHbox.ExpandFill` 被忽略
- 改为 `MarginContainer`，边距从 StyleBox 移到 MarginContainer 常量
- 清空/注入按钮现在正确右对齐

**2026-06-19 卡牌颜色系统 v1.4.0**
- 边框+左侧色条 = 英雄代表色（远古黑/无色深灰/英雄专属色/状态紫）

- 4 分类优先级：远古 > 无色 > 英雄 > 状态
- 角色筛选行新增「无色卡」「状态卡」按钮
- 角色 ID 全大写（IRONCLAD 而非 Ironclad），字典 key 需精确匹配

**2026-06-19 额外卡 SpinBox + 注入模式下拉**

- 新增注入模式下拉（额外添加/替换原卡）

**2026-06-19 遗物/药水稀有度筛选 v1.4.1**

## 5. UI 架构（当前）

```
CanvasLayer (SpawnCheatPanel, Layer=128)
  └── Control _rootPanel (840×600, 居中, ClipContents=false)
       ├── Panel bgPanel (FullRect, 金色描边背景, MouseFilter=Ignore)
       ├── MarginContainer (4px 边距 + 6px 底部)
       │   └── VBoxContainer (FullRect)
       │        ├── HBoxContainer _titleBar (32px, 可拖拽)
       │        ├── HBoxContainer _tabBar (24px, 手动 tab 按钮)
       │        ├── Control _tabContentArea (ExpandFill, ClipContents=true)
       │        │   └── CardTabContent / RelicTabContent / PotionTabContent / ResourceTabContent
       │        │       (FullRect 锚定, 只显示当前 active)
       │        └── Label 状态栏
       │
       └── [CardTabContent 内部]
            └── VBoxContainer (FullRect)
                 ├── HBoxContainer _charChips (角色筛选)
                 ├── HBoxContainer row2 (稀有度 + 搜索框)
                 ├── VBoxContainer contentVbox (ExpandFill)
                 │   ├── ScrollContainer _scroll (ExpandFill)
                 │   │   └── Control _cardGrid (手定定位卡片)
                 │   └── Panel bottomDock (34px, 计数/清空/注入)
                 └── (内嵌 vbox 防止 dock 溢出)
```
