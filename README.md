# sts2-Mods

xiatingfeng's Slay the Spire 2 MOD collection.

## MODs

### [隐身刷牌](./隐身刷牌) — Hidden Spawn Cheat / 隐藏生成作弊器

按下 F5 标记你想要的卡牌/遗物/药水，下次获得对应奖励时自动顶替。

- **Version**: 1.6.0
- **Platform**: Godot 4.6.2 Mono
- **Dependencies**: None (standalone)

### [地图胜率追踪](./地图胜率追踪) — Map Odds Tracker / 地图概率追踪

在爬塔过程中显示每个节点的遇敌概率。

- **Version**: 1.0.0
- **Platform**: Godot 4.6.2 Mono
- **Dependencies**: None (standalone)

### [模组同步检查器](./模组同步检查器) — Mod Sync Checker / 模组同步检测器

联机前快速对比 MOD 列表，支持导出/导入 MOD 配置文件。

- **Version**: 2.3.7
- **Platform**: Godot 4.6.2 Mono / BepInEx
- **Dependencies**: BaseLib (optional, soft dependency)

### [怪物行动预测](./怪物行动预测) — Monster Action Predictor / 怪物行动预测

预测怪物接下来 2 回合的行动意图，以半透明图标显示。

- **Version**: 1.0.3 (fork of [ByQsA/MonsterActionPredictor](https://github.com/aoyamaY/StS2-MonsterActionPredictor))
- **Platform**: Godot 4.6.2 Mono
- **Dependencies**: None (standalone)
